# PAC-MAN ULTIMATE

A classic Pac-Man game with AI opponents, handcrafted levels, minigames, and intense chiptune music!

## Features

- **Classic Pac-Man** — 5 handcrafted maze levels with unique colour themes
- **AI Ghosts** — Blinky, Pinky, Inky & Clyde with authentic chase behaviours
- **AI Rival Pac-Men** — 3 competitors (Speedy, Munchy, Chomper) racing you for dots
- **Ghost Tag Minigame** — play as a ghost chasing AI Pac-Men
- **Pellet Frenzy Minigame** — every dot is a power pellet, 6 ghosts, combo scoring
- **Leaderboard** — compete against AI rivals for the top score
- **Chiptune Music** — procedurally generated retro soundtrack
- **5 Themed Levels** — Classic (blue), Neon Arena (green), Shadow Maze (purple), Fire Fortress (orange), Ice Gauntlet (cyan)

## Requirements

- Python 3.8+
- pygame (`pip install pygame`)

## How to Play

```bash
python pacman.py
```

### Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move |
| Enter | Select |
| P | Pause |
| ESC | Menu / Back |
